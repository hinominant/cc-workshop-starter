# UI Component Patterns Dictionary

> Source: https://ui-design-dictionary.pages.dev/ (100 patterns, 15 categories)
> コンポーネント実装時の共通語彙と分類基準。

## Navigation (14)

| Component | 用途 |
|-----------|------|
| Hamburger Menu | 三本線アイコンでメニュー開閉（モバイル） |
| Tab Bar | 画面下部固定の主要セクション切替 |
| Breadcrumb | 階層パス表示 |
| Sidebar Navigation | 画面左側常時表示（管理画面定番） |
| Mega Menu | ホバーで大パネル展開 |
| Pagination | ページ番号リンク移動 |
| Infinite Scroll | スクロール末尾で自動追加 |
| Sticky Header | スクロール追従ヘッダー |
| Drawer | 画面端スライドインパネル |
| Command Palette | Cmd+K検索+コマンド実行 |
| Segmented Control | 2-5個の排他的ボタン切替 |
| Tabs | 水平タブでコンテンツ切替 |
| Stepper / Wizard | 複数ステッププロセス案内 |
| Anchor Navigation | ページ内スムーズスクロール |

## Layout (9)

| Component | 用途 |
|-----------|------|
| Grid Layout | 均等グリッドセル配置 |
| Masonry Layout | 高さ不均一のレンガ積み |
| Card Layout | 情報を独立カード単位で並べる |
| Split Screen | 画面左右2分割 |
| Hero Section | ページ冒頭の大ビジュアル+CTA |
| Bento Grid | 様々サイズのタイル状配置（モダン） |
| Holy Grail Layout | ヘッダー+フッター+3カラム |
| Full Bleed | 画面幅いっぱい没入型 |
| Sticky Sidebar | スクロール追従サイドバー |

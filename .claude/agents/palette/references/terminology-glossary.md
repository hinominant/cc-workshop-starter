# UI用語辞書テンプレート

> プロジェクト固有のUI用語を定義・統一するためのテンプレート。各プロジェクトで `.agents/glossary.md` として配置して使用する。

## 使い方

1. このテンプレートをプロジェクトの `.agents/glossary.md` にコピー
2. 「プロジェクト固有用語」セクションにプロジェクトの用語を追加
3. 「SaaSデフォルト用語」セクションで上書きが必要な用語を修正
4. UI実装時・ライティング時にこのファイルを参照

## 用語管理ルール

1. **新しいドメイン用語が出たら即座にこのファイルに追加する**
2. **同じ概念に複数の表記がある場合、1つに統一して他を「使わない」欄に記載**
3. **SaaSデフォルト用語をプロジェクトで上書きする場合、プロジェクト固有セクションに明記**
4. **UIに表示する文字列は、このファイルの日本語表記に従う**
5. **コード内の変数名・API名は、このファイルの英語表記に従う**
